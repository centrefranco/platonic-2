## Fixing Build Issues

When build issues arise on dev, test, or demo instances, always try to resolve them locally first before escalating.

```bash
# Builds and runs locally
npm run preview
```

Check to see if issues arise in the console. If so, they need to be fixed. If it builds correctly, reach out to the AT
team for help.
