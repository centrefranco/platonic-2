# Platonic

Platonic combines Astro and Decap into one unified and integrated platform, and includes examples of common uses cases and their best practices.

It is a synthesis of the best practices and lessons learned **so far**.

## Why Decap?

- No database needed: lower DevOps costs, simpler setup, no database migrations
- Git has built-in versioning for content recovery
- Editable website settings that are standardized across projects
- Easy collaboration with content creators, they just need a GitHub account
- Code-integrated data throughout the project lifecycle

## Why Astro?

- Raw HTML/CSS without JS wrapping
- Fewer abstraction layers (no more plugins for adding Google Analytics)
- Smart JS inclusion with Astro islands
- Blazingly fast with HTML-first static generation
